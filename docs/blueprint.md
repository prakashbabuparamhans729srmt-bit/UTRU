# **App Name**: Banana Buddy

## Core Features:

- Banana Fact Generator: Generate a fun and interesting fact about bananas.
- Recipe Suggestion: Suggest a simple recipe incorporating bananas. Uses an AI tool to find interesting recipes from an internal database of common meals and desserts. 
- Image Display: Display images and facts in an easy-to-read format.
- Recipe Display: Format and display the recipes to the user

## Style Guidelines:

- Primary color: Banana yellow (#FFE135) to represent the fruit.
- Background color: Pale yellow (#FAF2D9) as a soft, neutral backdrop.
- Accent color: Light green (#BDD871) as analogous color, creating a balanced design.
- Headline Font: 'Belleza', a humanist sans-serif appropriate to fashion and art; Body font: 'Alegreya' a humanist serif suitable for longer text
- Use cheerful icons of bananas and related food items.
- Gentle transitions for content loading and display.
